# 标签/分类封面图片

在此目录下放置与标签/分类同名的图片，即可在该标签/分类页面显示封面。

## 使用方法

1. **准备图片**
   - 支持格式：`.jpg`、`.jpeg`、`.png`、`.webp`
   - 建议尺寸：1920x1080 (16:9)
   - 文件名必须与标签/分类名称完全一致

2. **放置图片**
   ```
   static/images/terms/
   ├── 罗尔斯.jpeg
   ├── 学术之路.jpg
   ├── 哲学笔记.png
   └── AI.webp
   ```

3. **效果**
   - 访问 `/tags/罗尔斯/` 页面时，会显示 `罗尔斯.jpeg` 作为左侧封面
   - 如果图片不存在，则显示默认的标签页样式（无封面）

## 注意事项

- 文件名区分大小写
- 图片会自动裁剪和适配
- 支持多种格式，按优先级查找：jpg → jpeg → png → webp
- 可以同时存在多种格式，优先使用第一种找到的格式

## 优先级顺序

模板会按以下顺序查找图片：
1. `标签名.jpg`
2. `标签名.jpeg`
3. `标签名.png`
4. `标签名.webp`

## 示例

如果标签名为"罗尔斯"，会依次查找：
- `罗尔斯.jpg`
- `罗尔斯.jpeg` ✓（您的文件）
- `罗尔斯.png`
- `罗尔斯.webp`
